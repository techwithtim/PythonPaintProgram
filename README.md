# PythonPaintProgram

## Dependencies

- Python 3.6+

## Module Installation / Setup

```git clone <repo-url>```

```cd <repo-folder>```

```pip install -r requirements.txt```

or

```pip3 install -r requirements.txt```

or

```python - m pip install -r requirements.txt```

or

```python3 -m pip install -r requirements.txt```

## Running The Program

```python main.py```

or

```python3 main.py```
